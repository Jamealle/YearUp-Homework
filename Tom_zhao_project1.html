How does the Web work?


The Internet is a global system of interconnected computer networks that use the standard Internet protocol suite (TCP/IP) to link several billion devices worldwide.

What is HTML?


HTML is a set of code and tags that helps design and customize webpages.

Tags and Elements:

are tags or keywords that helps design the webpage. "!--....--" Defines a comment
"!DOCTYPE" Defines the document type
"a" Defines a hyperlink
"div" Defines a section in a document
"em" Defines emphasized text
"title" Defines a title for the document

Inline vs Block Elements

inline: is imputing CSS within the body of the html
EX: "style"
"body {background-color:lightgray}"

Block Elements: is a container for other HTML element.
EX: "div"
